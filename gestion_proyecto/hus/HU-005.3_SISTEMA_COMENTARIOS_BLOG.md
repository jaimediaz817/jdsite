## 🎯 DESCRIPCIÓN
Como visitante anónimo, puedo dejar un comentario público en cualquier artículo del blog, responder a comentarios existentes, y todo pasa por moderación antes de ser publicado. No se requiere registro ni cuenta de usuario.

---

## ✅ CRITERIOS DE ACEPTACIÓN
- [ ] Formulario público con campos: **Nombre (Obligatorio)**, Email (Opcional), Mensaje (Obligatorio)
- [ ] Límite de **1000 caracteres máximo** por comentario
- [ ] **Respuestas anidadas** hasta 2 niveles (respuesta directa solamente)
- [ ] TODO comentario nuevo entra en estado **PENDIENTE** por defecto
- [ ] Solo comentarios **APROBADOS** se muestran públicamente
- [ ] Auto moderación: No se permite links, html ni javascript
- [ ] Rate Limit: **3 comentarios cada 10 minutos por IP**
- [ ] Anti spam Honeypot nativo
- [ ] Avatar Gravatar automático por email
- [ ] Fecha relativa (hace X minutos)
- [ ] Notificación email al admin por cada comentario pendiente
- [ ] Panel completo de administración Django
- [ ] SEO Friendly: Todos los comentarios indexables, renderizados en servidor
- [ ] ✅ Funciona sin Javascript activado

5. [ ] Comentario vacio o con solo espacios
6. [ ] Email invalido en campo opcional
