## ✅ Descripcion
Como usuario leyendo un articulo del blog, quiero tener una barra de progreso discreta que me indique cuanto me falta por leer, sin que esta interfiera con mi experiencia de lectura ni robe atencion del contenido.

---

## ✅ Criterios de Aceptacion
- [x] Altura de la barra reducida a 2.5px
- [x] Fondo completamente transparente, sin color de fondo
- [x] Usar gradiente oficial del sitio: `linear-gradient(90deg, #7c3aed 0%, #ec4899 50%, #f43f5e 100%)`
- [x] Eliminar completamente todos los `!important` del CSS
- [x] Barra se muestra solo cuando se ha realizado mas del 5% del scroll
- [x] Barra se oculta automaticamente despues de 2 segundos sin actividad de scroll
- [x] Incluir efecto `backdrop-filter: blur(2px)` y opacidad base 0.85
- [x] Todas las transiciones con animacion suave de 200ms ease-out
- [x] No tapa ningun pixel del contenido del articulo
- [ ] Al alcanzar el 100% del articulo se muestra animacion de marca final
- [x] Funciona correctamente en todos los navegadores modernos
- [x] Responsivo en dispositivos moviles
- [x] No rompe ninguna funcionalidad existente
